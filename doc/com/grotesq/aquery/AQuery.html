Packagecom.grotesq.aquery
Classpublic final class AQuery
InheritanceAQuery Object

AS3용 Display List 도우미입니다. JS에서 document를 기준으로 DOM에 접근을 하듯, AS에서 stage를 기준으로 Display List를 탐색하는데 도움을 줍니다. Flash Professional에서 UI를 구성했을 때나, 부모 자식 관계가 복잡한 리스트를 가지고 작업할 때 도움이 됩니다.

initialize()method 
public static function initialize($stage:Stage):AQuery

AQuery를 초기화 합니다. AQuery.find 메소드를 전역적으로 실행할 때 문자열 형태의 표현식을 사용하기 위해서는 반드시 초기화를 해야 합니다. NativeWindow가 여러개 존재하는 AIR 애플리케이션의 경우 각각의 stage에 대한 낱개의 AQuery 객체를 생성해 사용할 수도 있습니다.

Parameters

$stage:Stage — stage 객체입니다.

Returns
AQuery

Example 
         import com.grotesq.aquery.AQuery;
         
         AQuery.initialize( stage );
         // or
         var $:AQuery = AQuery.initialize( stage );
